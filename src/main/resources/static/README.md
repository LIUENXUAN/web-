# 林间律动营地数字孪生平台

面向软件开发赛的营地数字孪生 Web 应用，包含作品展示页、Web 3D 模型查看器、前台业务应用、后台管理与 Node.js API 服务。

## 功能亮点

- 真实 SketchUp 营地模型转换为 Web 3D 展示。
- Three.js 模型查看器：旋转、缩放、平移、聚焦核心、全景完整、材质模式切换。
- 前台应用：营地区域、活动、预约、反馈等业务流程。
- 后台管理（最终版）：动画角色登录页、Bloom 液态玻璃风格运营后台、1 个核心仪表盘 + 9 大业务模块 + 3 级权限体系展示。
- 本地 JSON 数据库，便于比赛演示和代码审查。

## 本地运行

```bash
npm install
node server.cjs
```

浏览器打开：

```text
http://127.0.0.1:8787/
```

后台管理（最终版）：

```text
http://127.0.0.1:8787/admin/index.html
登录页为动画角色演示登录，输入任意密码即可进入最终版运营后台。
```

## 3D 模型说明

GitHub 仓库保留网页展示版模型：

- `assets/model/camp-real-web-nodraco.glb`
- `assets/model/camp-real-web.glb`

完整源模型、OBJ、BLEND、SKP 等超大文件已通过 `.gitignore` 排除，避免超过 GitHub 单文件限制。完整提交材料请使用比赛提交压缩包。

## 商业运营升级

本版本已从展示型网站升级为营地数字孪生运营管理平台，新增：

- 经营驾驶舱：营收、转化率、资产健康、告警闭环、客单价。
- 空间资产台账：帐篷、房车补给桩、护栏、照明、塔架等资产状态管理。
- 巡检任务：责任人、频次、截止日期、任务状态和巡检结果。
- 安全告警：风险等级、风险类型、处置状态和责任组。
- 业务闭环：游客预约 → 后台确认 → 订单支付状态 → 数据看板 → 巡检/安全处置。

专业架构说明见：`docs/系统架构升级方案.md`。
## 软件化控制台

新增 console/index.html 运营指挥舱，采用桌面软件式布局，支持 PWA 安装为本地应用。


## 最终版后台说明

- 入口：`http://127.0.0.1:8787/admin/index.html`
- 启动：在项目根目录运行 `node server.cjs`，或双击 `启动网站.bat`。
- 交付方式：后台已打包为 `admin/assets/` 下的静态文件，不依赖本机临时目录；拷贝整个 `website` 文件夹到其他电脑后执行 `npm install` 和 `node server.cjs` 即可访问。
- 登录：演示后台输入任意密码即可进入，便于比赛展示和离线演示。
