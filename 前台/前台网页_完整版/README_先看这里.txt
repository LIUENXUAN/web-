Nature Haven Camp 前台网页_完整版
================================

本目录包含两份内容：

1. 可直接运行版
   - 已构建好的正式前台页面。
   - 推荐打开方式：进入「可直接运行版」，双击「启动前台完整版.bat」。不要双击 index.html。
   - 浏览器会自动打开：http://127.0.0.1:6129/
   - 不要直接双击 index.html。直接用 file:// 打开会导致模块脚本、3D 模型或视频资源被浏览器限制，出现白屏。

2. 源码工程
   - React + Vite 源码。
   - 在有 Node.js 的电脑上可运行：
     npm install
     npm run dev
   - 或双击「启动源码开发版.bat」（首次仍需 npm install）。

便携说明：
- 可以把整个「前台网页_完整版」文件夹复制到其他电脑。
- 若目标电脑没有 Python/Node，也可以使用「可直接运行版」内置的 PowerShell 本地服务器脚本打开。
- 页面资源、图片、视频、3D 模型均已放在本目录内。

重要：
- 本目录是新增目录，不影响原来的「首页」文件夹。
