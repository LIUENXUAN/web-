营地数字孪生展示与运营管理平台使用说明

一、启动方式
双击“启动网站.bat”，系统会启动本地 Node 服务，并打开主展示页、前台应用、后台管理和 3D 模型页。

二、访问地址
主展示页：http://127.0.0.1:8787/index.html
前台应用：http://127.0.0.1:8787/app/index.html
后台管理（最终版）：http://127.0.0.1:8787/admin/index.html
3D模型页：http://127.0.0.1:8787/model-test.html

三、后台登录
最终版后台为演示登录页，输入任意密码即可进入。

四、比赛说明
本项目已从单纯展示网站升级为完整 Web 应用系统，包含前台业务、后台管理、Node.js 后端 API、JSON 数据库、Web 3D 模型展示、项目文档和技术报告。

五、3D 模型
默认加载真实 SKP 转换后的网页展示版，支持旋转、缩放、平移、视角切换和材质渲染模式；点击“完整模型”可加载完整 GLB。

六、自检脚本
node verify-model.cjs  检测 3D 模型加载
node verify-app.cjs    检测前台和后台页面


七、完整运行说明
把整个 website 文件夹复制到其他电脑后：
1. 安装 Node.js。
2. 在项目根目录执行 npm install。
3. 执行 node server.cjs，或双击“启动网站.bat”。
4. 访问 http://127.0.0.1:8787/admin/index.html 查看最终版后台。
